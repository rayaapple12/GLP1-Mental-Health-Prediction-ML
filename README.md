# GLP1 Mental Health Prediction (MSc Dissertation)

## Overview

This repository contains my MSc Artificial Intelligence dissertation project focused on predicting mental health outcomes using GLP-1 receptor genetic variants and phenotype data. The project explores machine learning techniques for identifying associations between genetic markers and mental health conditions.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset

This project was developed using a private genotype and phenotype dataset containing GLP-1 receptor genetic variants and associated clinical information.

The dataset cannot be shared publicly due to privacy, ethical, and data access restrictions.

Only the project methodology and code (where permitted) are included in this repository.

## Repository Contents

- Data preprocessing pipeline
- Machine learning model training
- Feature engineering
- Model evaluation
- Result visualization

*Raw research data is excluded from this repository.*

## Status

Project completed as part of my MSc in Artificial Intelligence at the University of Surrey.
